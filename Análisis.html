Datos

mañana <- c(4.0, 3.5, 3.8, 3.6, 4.1)
tarde  <- c(3.8, 3.5, 3.0, 2.9)
noche  <- c(3.1, 2.5, 2.6)

Explicación: Se definen tres vectores que representan los valores de cada turno.

datos <- data.frame(
  valor = c(mañana, tarde, noche),
  turno = c(rep("Mañana", length(mañana)),
            rep("Tarde", length(tarde)),
            rep("Noche", length(noche)))
)

Explicación: Se unifican los datos en una sola tabla, agregando una variable que identifica el turno.

Media por turno

datos %>%
  group_by(turno) %>%
  summarise(media = mean(valor))
# A tibble: 3 × 2
  turno  media
  <chr>  <dbl>
1 Mañana  3.8 
2 Noche   2.73
3 Tarde   3.3

Explicación: La media permite conocer el valor promedio en cada turno.

Varianza por turno

datos %>%
  group_by(turno) %>%
  summarise(varianza = var(valor))
# A tibble: 3 × 2
  turno  varianza
  <chr>     <dbl>
1 Mañana   0.0650
2 Noche    0.103 
3 Tarde    0.18

Explicación:
La varianza mide la dispersión de los datos respecto a la media.

Media general

media_total <- mean(datos$valor)
media_total
[1] 3.366667

Explicación: Representa el promedio global considerando todos los turnos.

Intervarianza (entre grupos)

intervarianza <- datos %>%
  group_by(turno) %>%
  summarise(
    n = n(),
    media = mean(valor)
  ) %>%
  mutate((n * (media - media_total)^2)) %>%
  summarise(intervar = sum(`(n * (media - media_total)^2)`)) %>%
  pull(intervar)

intervarianza
[1] 2.16

Explicación: Mide la variabilidad entre los promedios de cada turno.

Intravarianza (dentro de grupos)

intravarianza <- datos %>%
  group_by(turno) %>%
  summarise(sumatoria = sum((valor - mean(valor))^2)) %>%
  summarise(intra = sum(sumatoria)) %>%
  pull(intra)

intravarianza
[1] 1.006667

Explicación:
Mide la variabilidad dentro de cada grupo. Indica qué tan dispersos están los datos dentro de cada turno.

Conclusión

El análisis muestra diferencias entre los turnos en términos de promedio y variabilidad. El turno de la mañana presenta valores más altos en promedio, mientras que el turno de la noche presenta valores más bajos.

La intervarianza indica que existen diferencias entre los grupos, mientras que la intravarianza muestra la dispersión interna de cada uno. En conjunto, estos resultados permiten concluir que el turno influye en los valores observados.
